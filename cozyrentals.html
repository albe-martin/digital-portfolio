Cozy Rentals

Simple Rental Listing Platform

Cozy Rentals is a rental management web application built by a team of three. The goal of the project was to create a user-friendly platform where property listings could be managed and searched efficiently. With a MySQL database for storing listing information and user records, the platform supports basic user login, property browsing, and administrative controls. The front end was built using HTML and JavaScript, served via Apache.
