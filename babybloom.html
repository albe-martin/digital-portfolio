BabyBloom

Smart Baby Tracker Application

BabyBloom is a comprehensive baby tracking platform that allows parents to log and monitor daily activities like feeding, sleep, diaper changes, and growth metrics. Built using React, Express, and MySQL, it supports multi-user roles for parents and doctors. Features include daily logs, growth charts, reminders, and a beautiful UI optimized for mobile use.
